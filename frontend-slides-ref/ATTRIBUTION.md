# Attribution

The slide generation patterns in this toolkit are adapted from:

**frontend-slides** by zarazhangrui
https://github.com/zarazhangrui/frontend-slides

Specifically:
- `viewport-base.css` — responsive viewport-fitting patterns
- Style preset concept — curated visual themes over generic templates
- Single-file HTML architecture — zero-dependency presentations
- Animation patterns — entrance effects and background treatments

The original project is a Claude Code skill for generating animated HTML presentations.
Our adaptation focuses on workshop-specific features (polls, demos, agenda, cheatsheets).
